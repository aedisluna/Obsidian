## 🎯 Основы ArrayList

> [!info] ArrayList — это класс коллекции из пакета `java.util`, реализующий динамический массив, который автоматически расширяется при добавлении новых элементов.

### Ключевые моменты

- **ArrayList** — список элементов, допускает дубликаты и хранит элементы по порядку.
- **Динамическая** природа: размер увеличивается и уменьшается автоматически.
- **Тип элементов** — любой объект (через generic), например: `ArrayList<String>`, `ArrayList<Integer>`.[^1]

***

## ⚙️ Создание и использование

```java
import java.util.ArrayList;

// Создать пустой список строк
ArrayList<String> names = new ArrayList<>();

// Список чисел с начальной вместимостью
ArrayList<Integer> numbers = new ArrayList<>(50);
```

- `ArrayList<Type> имя = new ArrayList<>();` — объявление и создание.
- В Java 7+ можно не указывать тип справа (`<>`) при инициализации.

## 📁 Базовые операции

- `add(E e)` — добавить элемент.
- `get(int index)` — получить элемент по индексу.
- `set(int index, E e)` — заменить элемент.
- `remove(int index)` или `remove(Object o)` — удалить по индексу или по значению.
- `size()` — количество элементов.
- `clear()` — удалить все элементы.

  
Методы, которые есть у ArrayList (большинство является реализацией методов интерфейса Collection):

Методы

Описание

### Примеры

```java
ArrayList<String> list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.add("C++");

String second = list.get(1); // "Python"
list.set(2, "C#");           // заменили "C++" на "C#"
list.remove("Java");         // удалили "Java"
int count = list.size();     // 2
```

## 🌿 Итерация и обработка

- Самый удобный способ — через foreach:

```java
for (String item : list) {
    System.out.println(item);
}
```

- Можно использовать обычный цикл или итератор:

```java
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}
```

- Итераторы дают больше контроля и позволяют безопасно удалять элементы:

```java
Iterator<String> it = list.iterator();
while (it.hasNext()) {
    String val = it.next();
    if ("Python".equals(val)) {
        it.remove();
    }
}
```

## 💡 Полезные методы

- `contains(E e)` — проверяет наличие элемента.
- `isEmpty()` — пустой ли список.
- `indexOf(E e)` — первый индекс; `lastIndexOf(E e)` — последний.
- `toArray()` — преобразование к массиву.
- `subList(int from, int to)` — часть списка.

```java
if (list.contains("Java")) { ... }

String[] arr = list.toArray(new String);
```

## ⚔️ Частые ошибки и советы

- **NullPointerException** — если работаешь с неинициализированным списком.
- **IndexOutOfBoundsException** — неправильный индекс при работе с get/set/remove.
- ArrayList не потокобезопасен: если нужна работа из разных потоков — используй `Collections.synchronizedList` или другую коллекцию.
- Вставка и удаление элементов лучше делать в конце списка — операции в середине требуют сдвига остальных элементов.



***

## 🔥 Советы

- Если нужна постоянная длина — используй массив (`String[]`).
- Для быстрого поиска — рассматривай другие коллекции, например HashSet.
- ArrayList подходит для большинства случаев, если часто нужна динамика размеров.
- При большом числе вставок/удалений в начало или середину — рассмотрей LinkedList.
