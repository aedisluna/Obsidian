
## 🧭 Навигация по коду

### Переходы между файлами

```
Ctrl + N            - Поиск класса
Ctrl + Shift + N    - Поиск файла
Ctrl + Alt + Shift + N - Поиск символа
Ctrl + E            - Недавние файлы
Ctrl + Shift + E    - Недавно редактируемые
Ctrl + Tab          - Переключение между вкладками
Ctrl + F12          - Структура файла
```


### Навигация внутри файла

```
Ctrl + G            - Перейти к строке
Ctrl + F            - Поиск в файле
Ctrl + R            - Замена в файле
Ctrl + Shift + F    - Поиск во всем проекте
Ctrl + Shift + R    - Замена во всем проекте
F3 / Shift + F3     - Следующее/предыдущее вхождение
```


### Переходы к определениям

```
Ctrl + B или Ctrl + Click - Перейти к объявлению
Ctrl + Alt + B      - Перейти к реализации
Ctrl + Shift + B    - Перейти к объявлению типа
Ctrl + U            - Перейти к суперметоду/суперклассу
Alt + F7            - Найти использования
Ctrl + Alt + F7     - Показать использования
```


## ✏️ Редактирование

### Базовое редактирование

```
Ctrl + D            - Дублировать строку
Ctrl + Y            - Удалить строку
Ctrl + X            - Вырезать строку (если ничего не выделено)
Ctrl + Shift + Up/Down - Переместить строку вверх/вниз
Ctrl + Shift + U    - Переключить регистр
Alt + J             - Выделить следующее вхождение слова
Alt + Shift + J     - Убрать выделение последнего вхождения
Ctrl + Alt + Shift + J - Выделить все вхождения
```


### Множественные курсоры

```
Alt + Click         - Добавить курсор
Alt + Shift + Click - Создать прямоугольное выделение
Ctrl + Alt + Shift + Click - Колонка курсоров
```


### Автодополнение и генерация

```
Ctrl + Space        - Базовое автодополнение
Ctrl + Shift + Space - Умное автодополнение
Ctrl + Alt + Space  - Автодополнение класса
Alt + Insert        - Генерация кода
Ctrl + I            - Реализовать методы
Ctrl + O            - Переопределить методы
Ctrl + Alt + T      - Окружить с... (try-catch, if, etc.)
```


## 🔧 Рефакторинг

### Основной рефакторинг

```
Shift + F6          - Переименовать
Ctrl + F6           - Изменить сигнатуру метода
Ctrl + Alt + M      - Извлечь метод
Ctrl + Alt + V      - Извлечь переменную
Ctrl + Alt + C      - Извлечь константу
Ctrl + Alt + F      - Извлечь поле
Ctrl + Alt + P      - Извлечь параметр
Ctrl + Alt + N      - Встроить (inline)
F6                  - Переместить
Ctrl + Shift + F6   - Изменить тип
```


## 🐛 Отладка

### Точки остановы и выполнение

```
Ctrl + F8           - Переключить точку останова
Ctrl + Shift + F8   - Показать все точки останова
F7                  - Шаг с заходом
F8                  - Шаг без захода
Shift + F8          - Шаг наружу
F9                  - Продолжить выполнение
Alt + F9            - Выполнить до курсора
Ctrl + Shift + F9   - Компилировать и запустить
```


## 🔍 Поиск и замена

### Расширенный поиск

```
Double Shift        - Поиск везде
Ctrl + F            - Поиск в файле
Ctrl + Shift + F    - Поиск в проекте
Ctrl + R            - Замена в файле
Ctrl + Shift + R    - Замена в проекте
Alt + F3            - Поиск слова под курсором
Ctrl + F3           - Поиск выделенного текста
```


## 🎯 Специальные функции

### Работа с кодом

```
Ctrl + Alt + O      - Оптимизировать импорты
Ctrl + Alt + L      - Форматировать код
Ctrl + Shift + J    - Объединить строки
Ctrl + /            - Закомментировать строку
Ctrl + Shift + /    - Блочный комментарий
Ctrl + Q            - Показать документацию
Ctrl + P            - Показать параметры метода
Ctrl + Shift + I    - Быстрое определение
```


### Работа с окнами

```
Alt + 1             - Окно Project
Alt + 9             - Окно Version Control  
Alt + 6             - Окно TODO
Alt + 4             - Окно Run
Alt + 5             - Окно Debug
Shift + Escape      - Закрыть активное окно инструментов
Ctrl + Shift + F12  - Максимизировать редактор
```
