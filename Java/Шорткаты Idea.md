# Шорткаты IntelliJ IDEA (Windows/Linux)

## Навигация и поиск

- Ctrl + N — открыть класс по имени.
- Ctrl + Shift + N — открыть файл по имени.
- Shift Shift — универсальный поиск (Search Everywhere).
- Ctrl + E — недавно открытые файлы.
- Ctrl + Shift + E — недавно редактированные места (Recent Locations).
- Ctrl + B / Ctrl + Click — перейти к объявлению.
- Ctrl + Alt + B — перейти к реализации.
- Alt + F7 — найти использования.
- Ctrl + Alt + ←/→ — назад/вперёд по истории.
- Ctrl + F12 — структура файла (popup).
- Alt + 7 — открыть окно Structure.
- Ctrl + Shift + A — поиск команд/настроек (Find Action).
- Ctrl + Shift + Backspace — перейти к последнему изменению.
- F2 / Shift + F2 — к следующей/предыдущей ошибке.


## Редактирование и код

- Alt + Enter — быстрые фиксы и намерения.
- Ctrl + / — комментировать строку; Ctrl + Shift + / — блок.
- Ctrl + D — дублировать строку/блок.
- Ctrl + Y — удалить строку.
- Ctrl + W / Ctrl + Shift + W — расширить/сузить выделение.
- Ctrl + Alt + L — форматировать код.
- Ctrl + Alt + O — оптимизация import.
- Ctrl + Space — базовое автодополнение.
- Ctrl + Shift + Space — умное автодополнение.
- Ctrl + P — подсказка параметров метода.
- Ctrl + Q — быстрая документация.
- Ctrl + Shift + Enter — завершить конструкцию (вставить недостающие скобки/точку с запятой).
- Ctrl + Shift + J — склеить строки (Join Lines).
- Ctrl + / и Ctrl + Shift + / — комментарии (повторно, для акцента).


## Рефакторинг

- Shift + F6 — переименование.
- Ctrl + Alt + V — выделить переменную.
- Ctrl + Alt + P — выделить параметр.
- Ctrl + Alt + M — выделить метод.
- Ctrl + Alt + F — выделить поле.
- Ctrl + Alt + C — выделить константу.
- Ctrl + Alt + Shift + T — меню Refactor This.


## Генерация и Live Templates

- Alt + Insert — генерация кода (конструкторы, getters/setters, toString).
- Ctrl + J — список Live Templates.
- Tab — развернуть шаблон.


## Поиск и замена в тексте

- Ctrl + F — поиск в файле.
- Ctrl + R — замена в файле.
- Ctrl + Shift + F — поиск по проекту.
- Ctrl + Shift + R — замена по проекту.


## Запуск и отладка

- Shift + F10 — запуск.
- Shift + F9 — отладка.
- Ctrl + F2 — остановить.
- F8 / F7 — Step over / into.
- Shift + F8 — Step out.
- Alt + F9 — Run to cursor.
- Alt + F8 — Evaluate Expression.
- Ctrl + F8 — поставить/снять breakpoint.


## Работа с окнами и панелями

- Alt + 1 — Project; Alt + 4 — Run; Alt + 5 — Debug; Alt + 6 — Problems; Alt + 9 — Git.
- Alt + F12 — терминал; Alt + 0 — Messages.
- Ctrl + Shift + F12 — максимум места для редактора/окна.
- Esc — фокус в редактор.
- Ctrl + ` — Quick Switch Scheme (темы/кодстайл/кеймап).
- Ctrl + Alt + S — Settings; Ctrl + Alt + Shift + S — Project Structure.


## Навигация по вкладкам и местам

- Ctrl + Tab — переключатель между файлами/окнами.
- Ctrl + PageUp/PageDown — соседние вкладки.
- Ctrl + F3 — найти слово под кареткой.


## Полезные мелочи

- Ctrl + Alt + Y — синхронизировать с диском.
- Alt + Shift + I — Inspect Code для текущего файла.
- Ctrl + S — сохранить всё.
