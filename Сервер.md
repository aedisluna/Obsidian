## tmux (сессии)

1. Установи tmux на сервере:
  ```bash
  sudo apt install tmux
  ```
  
  2. Создай сессию и запусти сервер внутри неё:
```bash
cd ~/minecraft-server tmux new -s mc # внутри tmux: ./run.sh   # или твоя java-команда
```
Теперь сервер бежит внутри tmux.​

3. **Отсоединиться, но не останавливать сервер:** 
 - Нажми `Ctrl+B`, затем `D` (detach).[reddit+1](https://www.reddit.com/r/MinecraftServer/comments/17b2new/server_crashes_when_i_close_sshconnection/)​    
 - SSH-сессию можно закрывать, сервер продолжит работать.
 
3. **Подключиться к консоли позже:**
```bash
tmux attach -t mc
```
или, если одна сессия:
```bash
tmux a
```

5. Внутри tmux консоль ведёт себя как обычно: пишешь `stop`, видишь логи и т.п.

## Подключение к серверу с Windows

### Вариант A: через терминал (PowerShell / cmd)

1. На своём ПК открой **PowerShell**.
2. Введи: 
```powershell
ssh ИМЯ_ПОЛЬЗОВАТЕЛЯ@192.168.0.112
```
Например:
```powershell
ssh funeralparty@192.168.0.112
```

3. При первом подключении согласись с fingerprint (`yes`), введи пароль.[linuxbabe+1](https://www.linuxbabe.com/linux-server/ssh-windows)​
