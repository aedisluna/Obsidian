[[Базы данных]]

**Первичный ключ**
Поле, значение которого однозначно определяет запись в таблице

**Внешний ключ**
Поле, которое ссылается на первичный ключ в другой таблице

- таблица с внешним ключом называется *дочерей*, а с первичным *родительской*
- при создании записей в дочерей таблице, значение поля внешнего ключа есть в родительской
- наличие первичных и внешних ключей не обязательно

**Вывод текста**
`SELECT "Hello world"`
Выведет Hello world

**AS**
`SELECT member_ID, member_name AS name FROM x`
`SELECT member_ID, member_name  name FROM x`

Присвоит полю member_name псевдоним name

**Литералы**
Указанное явным образом фиксированное значение

- строковой
	- последовательность символов, заключённая в кавычки (одинарные или двойные)
	- поддерживает специальные последовательности, начинающиеся с `\`, такие как `\n` для новой строки
- числовой
	- целые и дробные числа, разделитель дроби - точка
	- можно только дробную часть
	- положительное и отрицательное
	- экспонента
	- так же доступны операторы арифметики
		- `% или MOD` деление по модулю, выводит остаток
		- `*` умножение
		- `+` сложение
		- `-` вычитание
		- `/` деление
		- `DIV` целочисленное деление
- логический
	- TRUE FALSE
- NULL
	- Отсутствие значения
- литерал даты и времени
	- YYYY-MM-DD с любыми знаками препинания в качестве разделителей
	- hh-mm-ss
	- YYYYMMDDhhmmss
