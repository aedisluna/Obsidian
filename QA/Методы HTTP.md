[[Бэкэнд]] [[HTTP]]

| Метод   | Scope      | Пояснение                           |
| ------- | ---------- | ----------------------------------- |
| GET     | Collection | Возвращает все ресурсы в коллекции  |
| GET     | Resource   | Возвращает ресурс                   |
| HEAD    | Collection | Возвращает хедеры всех ресурсов     |
| HEAD    | Resource   | Возвращает хедер ресурса            |
| POST    | Collection | Создаёт ресурс                      |
| PUT     | Resource   | Обновляет ресурс                    |
| PATCH   | Resource   | Обновляет ресурс                    |
| DELETE  | Resource   | Удаляет ресурс                      |
| OPTIONS | Any        | Возвращает доступные методы и опции |
