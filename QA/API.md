[[Бэкэнд]]

Application programming interface

Определенный набор правил, протоколов и инструментов,позволяющий различному по взаимодействовать между собой. Определяет способы, с помощью которых приложения могут отправлять запросы друг другу для получения данных или выполнения операций. Позволяет разным системами обмениваться информацией и фукнкциональностью без раскрывать внутреннюю реализацию
