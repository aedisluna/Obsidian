[[Бэкэнд]] [[HTTP]]

**TCP (Transmission Control Protocol)**: TCP является протоколом транспортного уровня в модели OSI. Он обеспечивает надежную и устойчивую передачу данных между устройствами. TCP разделяет данные на пакеты, нумерует их и гарантирует их доставку в правильном порядке. Он также обеспечивает контроль ошибок и управление потоком данных.

**UDP (User Datagram Protocol):** UDP также является протоколом транспортного уровня и обеспечивает более легковесный способ передачи данных. В отличие от TCP, UDP не гарантирует надежную доставку данных и не обеспечивает контроль ошибок и управление потоком. Он часто используется для передачи данных, где небольшая потеря пакетов приемлема, например, в потоковом видео или в онлайн-играх.

**HTTP (Hypertext Transfer Protocol):** HTTP является протоколом прикладного уровня, используемым для передачи данных в вебе. Он базируется на протоколе TCP и определяет стандарты для запросов и ответов между веб-браузерами и веб-серверами. HTTPS (HTTP Secure) - это версия HTTP, которая добавляет шифрование с использованием SSL/TLS для обеспечения безопасной передачи данных.

 **SMTP (Simple Mail Transfer Protocol):** SMTP является протоколом прикладного уровня, используемым для отправки электронной почты между серверами. Он определяет правила для отправки и приема почтовых сообщений.

**POP3 (Post Office Protocol version 3):** POP3 также является протоколом прикладного уровня для работы с электронной почтой. Он используется для загрузки почтовых сообщений с сервера на локальный компьютер.

**IMAP (Internet Message Access Protocol):** IMAP также является протоколом для работы с электронной почтой. В отличие от POP3, IMAP хранит почтовые сообщения на сервере, что позволяет клиентам управлять почтовым ящиком с разных устройств.

**DNS (Domain Name System)**: DNS является протоколом, используемым для преобразования доменных имен (например, [www.example.com](http://www.example.com/)) в IP-адреса, необходимые для определения местонахождения серверов в сети.

**SNMP (Simple Network Management Protocol):** SNMP - это протокол управления сетевыми устройствами. Он позволяет удаленно управлять и мониторить сетевое оборудование, такое как маршрутизаторы и коммутаторы.
